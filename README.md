# Music Visualizer

p5.js와 p5.sound.js를 활용한 음악 반응형 비주얼라이저  
오디오의 음량 및 주파수 데이터를 분석하여, 밤 드라이브 풍경이 음악에 맞춰
실시간으로 변화하도록 구현했습니다.

## 주요 기능

- 화면 클릭 시 음악 로드 및 재생
- p5.Amplitude / p5.FFT를 이용한 오디오 분석
- 음악에 따라 도로, 자동차, 가로등, 하늘 색상이 실시간으로 변화
- 저음/중음/고음에 각각 다른 시각 요소 매핑
- 키보드 입력을 통한 재생 속도 조절
- 곡 진행에 따라 밤 → 새벽 → 낮으로 전환되는 배경 연출

## 기술 스택

- p5.js – 그래픽 및 애니메이션
- p5.sound.js – 오디오 로드 및 분석 (Amplitude, FFT)

## 사용 방법

1. **재생**
   - 화면을 클릭하면 음악과 함께 비주얼라이저가 시작됩니다
2. **재생 / 일시정지**
   - 마우스 클릭으로 음악 재생 및 정지
3. **재생 속도 조절**
   - ↑ / ↓ : 미세 조절
   - ← / → : 빠른 조절  
   - 재생 속도 범위: 0.5x ~ 2.0x

## 커스터마이징

### 음악 파일 변경
```javascript
function preload() {
  song = loadSound('your-music.mp3');
}
```

### 캔버스 크기 조절

```javascript
createCanvas(800, 600);
```

### 하늘 색상 조절

```javascript
skyColors.night = {
  top: [15, 10, 40],
  bottom: [40, 20, 60]
};
```

## 파일 구조

```
project/
├── index.html        # 실행용 HTML
├── sketch.js         # 메인 비주얼라이저 코드
└── blinding_lights.mp3  # 음악 파일
```

## 주의사항

- mp3 파일은 sketch.js와 동일한 디렉토리에 있어야 합니다
- 브라우저 정책상 사용자 클릭 이후에만 사운드가 재생됩니다
- 디오 분석 결과에 따라 시각 효과가 실시간으로 변합니다
